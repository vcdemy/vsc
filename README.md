## 課程內容

* 安裝 [Visual Studio Code](https://code.visualstudio.com/)
* 介面介紹
* 安裝 Extensions
* 編寫 HTML, CSS (使用 emmet)
* 編寫 Markdown
* 編寫 Python
* 使用 git
* 實作練習
    * 使用 Markdown 簡單寫個 readme
    * 使用 Emmet 簡單寫一個網頁
    * 使用 git 將寫好的東西推上 github
